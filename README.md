# Breast Cancer Prediction with DecisionTree

## About the Project

This project aims to predict breast cancer outcomes using a decision tree. The dataset used for this project is the [Breast Cancer Wisconsin (Diagnostic) Data Set](https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic), containing features to prediction the class : Malignant(+ve) or Benign(-ve).
